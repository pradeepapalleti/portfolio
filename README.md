# Pradeepa Palleti Portfolio

A personal portfolio website for Pradeepa Palleti, built with HTML, CSS, and JavaScript.

## Live Site

- Deployed link: [palletipradeepa.vercel.app](https://palletipradeepa.vercel.app)

## Project Structure

- `index.html` - main entry point
- `style.css` - site styling
- `script.js` - interactive behavior
- `profile.jpeg` - profile image
- `PALLETI PRADEEPA - CSE.pdf` - resume download asset
- `project-snapshots/` - screenshots used in the project modal

## Run Locally

1. Clone the repository.
2. Open `index.html` in a browser, or use a local static server.

## Notes

- The portfolio is designed as a single-page site with sections for About, Skills, Projects, Experience, and Contact.
- Project cards include live links and GitHub links where available.
